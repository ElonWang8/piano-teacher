# 六个新功能 — 设计规格

## 1. 上课记录一键复制

课程记录卡片加「📋 复制」按钮。点击生成文字摘要并复制到剪贴板：

```
【PianoRecord 上课记录】
学生：王小明
日期：2026年5月11日 周六 09:00-09:45
曲目：拜厄 No.45、哈农 No.3
掌握情况：手指独立性有进步，注意左手伴奏力度控制
本周作业：哈农 No.4 慢练、拜厄 No.46 预习
```

Toast 提示"已复制，可粘贴发送给家长"。

实现：纯前端 `navigator.clipboard.writeText()`。无需 API。

## 2. 曲目库快捷输入

课程记录表单的曲目录入框下方添加曲目标签栏：

预置标签：小汤1/小汤2/小汤3、拜厄、车尔尼599/849/299、哈农、音阶、其他

点击标签追加到输入框（逗号分隔）。也可手动自由输入。

标签存代码常量，无需数据库。

## 3. 月度学生报表

看板新增「月报」Tab 或子页面。按月选择（默认本月），显示表格：

| 学生 | 上课次数 | 请假/旷课 | 课时消耗 | 缴费金额 | 剩余课时 | 出勤率 |

数据来源：Lesson + Payment + Student 表。

支持复制为文本或导出。

## 4. JSON 备份 + NAS 定时备份

**手动备份：** 设置页加「导出数据」按钮，生成 JSON 文件下载。包含所有表数据。

API：`GET /api/backup` — 返回 JSON，前端触发下载。

**定时备份：** Docker compose 加 sidecar 容器，每天凌晨用 `curl` 调用 API 保存到 NAS 目录。

## 5. 学生导入导出

**导入：** 设置页加「导入学生」按钮，支持 .xlsx 和 .json 上传。

Excel 模板：姓名、年龄、级别、家长手机号、状态、备注。

**导出：** 学生列表页加「导出」按钮，下载 Excel。

使用 `xlsx` 库处理 Excel。

## 6. 日历节假日显示

在排课日历中显示中国法定节假日。

方案：项目内置 `src/lib/holidays.ts`，存 2026-2027 年法定假日日期数组。日历网格中日假期高亮（红色日期 + "假" 标签）。调休上班日也可标记。

节假日数据：手动维护，每年更新一次。来源：国务院办公厅放假通知。

日历中，假日日期格显示红色，鼠标悬停显示假日名称（如"劳动节"）。

## 实现策略

功能 1-3、6：纯前端/小改动，可并行
功能 4：新 API + Docker sidecar
功能 5：新 API + 新依赖（xlsx）
