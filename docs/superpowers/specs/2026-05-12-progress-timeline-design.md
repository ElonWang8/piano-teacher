# 学生学琴历程 — 设计规格

## 概述

在学生详情页「学习进度」Tab 中新增学琴历程时间线，替代现有的简单数字展示。包含进度概览卡片、学习阶段分段、按年分组的时间线。

## 方案

改造 `src/app/students/[id]/page.tsx` 的「学习进度」Tab 内容。

### 1. 进度概览卡片（3 个）

| 卡片 | 数据来源 | 计算方式 |
|------|----------|----------|
| 累计课时 | Lesson 表 | `COUNT(*) WHERE status='ATTENDED'` |
| 学完曲目 | repertoire 字段 | 统计有曲目内容的 Lesson 去重 |
| 学琴时长 | created 字段 | `MAX(date) - MIN(date)` 按月/年显示 |

### 2. 学习阶段条

基于 Student 表的 `level` 字段和 Lesson 表时间范围，自动划分：
- 入门（小汤1-3）
- 初级（拜厄、车尔尼599）
- 中级（车尔尼849、小奏鸣曲）
- 高级（车尔尼299+）

每阶段显示：阶段名、持续时间、曲目数。
背景色：绿、蓝、橙、红，白色文字。

### 3. 学琴历程时间线

按年份/月份分组展示所有课程记录：
- 竖线 + 圆点节点
- 每年一个标题节点（大圆点，强调色）
- 每月/每次课一个子节点（小圆点）
- 显示：日期、曲目、备注摘要

## 数据来源

从 `GET /api/students/[id]` 返回的 Lesson 数组提取。无需新增 API。

## 技术实现

- 纯前端计算（分组、聚合）
- 新增 `src/components/students/progress-timeline.tsx` 组件
- 在学生详情页「学习进度」Tab 中引入

## 移动端适配

- 阶段条 `flex-col` 纵向排列
- 卡片 `grid-cols-2` 手机 2 列
- 时间线保持不变（竖线在小屏也清晰）

## 测试

- 单元测试：时间分组、曲目计数逻辑
- 组件测试：空数据、只有 1 条数据、跨年数据
